# Virtual Library System

This project was completed as part of a programming assignment. 
The goal was to practice working with Python data structures, functions, and basic object-oriented concepts by implementing a simple library management workflow.

# Features
Add and manage books in a virtual library
Track borrowed books
Search for books by title or author
Store and update book information

# Technologies Used
Python
Jupyter Notebook

# Files
Virtual_Library.ipynb – core library management functionality
book_tracker.ipynb – simple book tracking logic

# What I Practiced
Working with Python classes and functions
Using lists and dictionaries to store structured data
Basic debugging and program organization
